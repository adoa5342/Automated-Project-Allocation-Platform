# Test Configuration and Usage

## Running Tests

Install dependencies:
```bash
pip install -r requirements.txt
```

Run all tests:
```bash
pytest
```

Run tests with coverage:
```bash
pytest --cov=allocator --cov-report=html
```

Run specific test file:
```bash
pytest tests/test_schema.py
pytest tests/test_algorithm.py
pytest tests/test_app.py
pytest tests/test_performance.py
```

## Test Coverage

The test suite aims for 80%+ coverage and includes:

- **Schema tests** (`test_schema.py`): Validates Pydantic models, field constraints, and data validation
- **Algorithm tests** (`test_algorithm.py`): Tests scoring functions, allocation logic, and constraints
- **API tests** (`test_app.py`): Tests FastAPI endpoints, request/response handling, and integration
- **Performance tests** (`test_performance.py`): Tests execution speed with various dataset sizes

Coverage reports are generated in `htmlcov/` directory.

## Performance Testing

Test speed with larger datasets

### Test Categories

#### Basic Tests (Default)
These tests run by default and complete quickly:

- `test_small_dataset_baseline`: 10 groups, 20 projects
- `test_medium_dataset`: 50 groups, 100 projects  
- `test_scaling_performance`: Tests multiple sizes (10-100 groups)
- `test_constrained_vs_unconstrained`: Compares performance with/without skill constraints

#### Slow Tests
Marked with `@pytest.mark.slow`:

- `test_large_dataset`: 200 groups, 300 projects

#### Stress Tests  
Marked with `@pytest.mark.stress`:

- `test_stress_dataset`: 500 groups, 500 projects

### Running Performance Tests

```bash
# Run all basic performance tests (fast)
pytest tests/test_performance.py -v -s

# Run basic tests only (exclude slow and stress)
pytest tests/test_performance.py -v -s -m "not slow and not stress"

# Run including slow tests
pytest tests/test_performance.py -v -s -m "not stress"

# Run all tests including stress tests
pytest tests/test_performance.py -v -s

# Run without coverage (faster)
pytest tests/test_performance.py -v -s --no-cov
```

## Cleaning Test Artifacts

To remove files generated by pytest and pytest-cov:

```bash
rm -rf .pytest_cache .coverage htmlcov
```
